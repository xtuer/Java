**核心技术:**

* Spring
* SpringMVC
* SpringSecurity
* MyBatis
* logback
* jQuery
* Bootstrap
* 分页使用 simple pagination，参考 <http://qtdebug.com/fe-pagination/>

**注意:**

只对页面的 URL 做了权限校验，对 resources 的 URL 没有做权限校验，如有必要，在 spring-security.xml 中进行配置。
